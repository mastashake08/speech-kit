# Features
- Speak Commands
- Listen for voice commands
- Add your own grammar
- Transcribe words and output as file.

```
npm install @mastashake08/speech-kit
```

### Import
```
import SpeechKit from '@mastashake08/speech-kit'
```

### Instantiate A New Instance
```
new SpeechKit()
```
